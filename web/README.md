# Web

This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).

## Usar

Run (development):

```bash
yarn install
yarn run --proxy localhost:8080
```

**Se debe tener el servidor corriendo**


## Compilar

```bash
yarn build
```

Se debe servir la carpeta `build` como si estuviese en el _root_ (`/`). Todas las webs que se devuelvan deben responder con `index.html`.
